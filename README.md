# Multi-Region Sales Variance Report (Excel)

This project demonstrates an Excel-based sales variance analysis across multiple regions and quarters.

## Key Features
- Consolidated quarterly sales data from multiple regions
- Calculated variance against sales targets and previous periods
- Applied conditional formatting to highlight over- and under-performance
- Created pivot tables for region-wise and quarter-wise analysis
- Designed an Excel dashboard with charts for reporting

## Tools Used
- Microsoft Excel
